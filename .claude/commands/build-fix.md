# Build Fix

Flutter のビルドエラーと静的解析エラーを段階的に修正してください。

## 手順

1. `flutter analyze` を実行してエラーを確認
2. エラーを重要度順にソート（error > warning > info）
3. **1つずつ** エラーを修正
4. 各修正後に `flutter analyze` を再実行して検証
5. すべてのエラーが解消されたら `flutter test` を実行

## ルール

- 最小限の差分で修正すること
- アーキテクチャ変更やリファクタリングは行わない
- 既存のプロジェクトパターンに従う
- 修正が不明な場合は停止してユーザーに確認
